### A. Karakteristik Doubly Linked List (DLL) 
Doubly Linked List (DLL) merupakan struktur data linear di mana setiap elemen atau node memiliki koneksi dua arah. Berbeda dengan tipe singly yang searah, setiap node pada DLL memiliki dua pointer, yaitu Next untuk mengakses elemen sesudahnya dan Prev untuk kembali ke elemen sebelumnya. Struktur ini dikendalikan oleh dua pointer utama: First sebagai penanda ujung depan dan Last sebagai penanda ujung belakang, yang secara signifikan mempermudah navigasi data.

### 1. Mekanisme Pencarian (Searching)
Operasi pencarian dilakukan untuk mengidentifikasi keberadaan atau posisi data tertentu di dalam rangkaian. Hal ini dicapai melalui teknik traversal (penelusuran), di mana sistem akan memeriksa isi setiap node satu per satu secara berurutan. Proses ini akan otomatis berhenti begitu kriteria data yang dicari telah terpenuhi.

### 2. Penghapusan (Delete)
Prosedur penghapusan elemen pada Doubly Linked List dirancang secara spesifik untuk menangani berbagai posisi node guna menjaga integritas hubungan dua arah antar-elemen. Operasi ini mencakup DeleteFirst yang berfungsi untuk memutus dan menghapus elemen pada pangkal daftar, serta DeleteLast yang digunakan untuk membuang elemen di posisi paling ujung. Selain itu, terdapat pula mekanisme DeleteAfter yang memungkinkan penghapusan elemen di posisi tengah dengan cara memutus hubungan pada node yang terletak tepat setelah referensi posisi tertentu.

void deleteFirst(linkedlist &List){
    if(isEmpty(List) == true){
        cout << "List kosong! Tidak ada node yang dihapus." << endl;
        return;
    }
    address pDel = List.first;
    if(List.first == List.last){
        List.first = List.last = Nil;
    } else {
        List.first = pDel->next;
        List.first->prev = Nil;
        pDel->next = Nil;
    }
    pDel->prev = Nil;
    cout << "Node " << pDel->isidata.nama << " berhasil dihapus dari posisi pertama." << endl;
    dealokasi(pDel);
}

void deleteLast(linkedlist &List){
    if(isEmpty(List) == true){
        cout << "List kosong! Tidak ada node yang dihapus." << endl;
        return;
    }
    address pDel = List.last;
    if(List.first == List.last){
        List.first = List.last = Nil;
    } else {
        List.last = pDel->prev;
        List.last->next = Nil;
        pDel->prev = Nil;
    }
    pDel->next = Nil;

    if(Prev->next == List.last){

    address pDel = nodeNext->prev;
    nodeNext->prev = pDel->prev;
    pDel->prev->next = nodeNext;
    pDel->next = Nil;
    pDel->prev = Nil;

        deleteFirst(List);
    } else if(target == List.last){
        deleteLast(List);
    } else {

        R->prev = L;

}
